# General

This directory contains the files and instructions needed to build the irradiation chamber

`aperture` contains the CAD and STL files for the needed apertures

`assambly` contains the instructions to build the chamber and a list of all needed parts

`holder` contains the CAD and STL files for the needed holder

`other` contains the CAD and STL files for the needed other 3D printed parts, as well as a ocitrue and description of the double walled beaker

`walls` contains the CAD and STL files for the needed walls
