# General

This directory contains labjournal protocols related to irradiation setup.

`AE-515` contains info about procedure for 3D printing of components of advanced irradiation setup,

`AE-516`contains protocol for design and printing of holder for 4-neck photoreactor (BOLA GL14 cap for PT1000 sensor),

`AE-541` contains info about design and printing of protective cover of H2 sensor,

`AE-594` contains info about design and printing of holder for power measurements
