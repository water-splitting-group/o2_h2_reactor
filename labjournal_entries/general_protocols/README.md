# General

This directory contains labjournal entries of the general procedures used in work.

`firesting_software` contains info about procedure for in situ O2 measurements using Pyroscience Workbench software,

`gas_phase_h2_calib` contains info about procedure of H2 sensor calibration in gas phase,

`hydrogen` contains info about the procedure of getting and transfer of H2 gas using balloon,

`in-situ_h2_o2_measurement` contains info about main steps for simultaneous in situ O2 and H2 measurements in photocatalytic tests,

`liquid_phase_h2_calib` contains info about procedure of H2 sensor calibration in liquid phase,

`power_determination` contains info about determination of the required analogue setting of power source for desired power output,

`settings_3d_printer` contains overview of standard settings for 3D printer,

`use_3d_printer` contains the procedure of 3D printing of the desired model
