# General

This directory contains labjournal entries of the used devices.

`3d_printer` contains info about model of the used 3D printer,

`18a_power_source` contains info about power supply,

`365_uhp_led` contains info about 365 nm LED,

`firesting_o2_meter` contains info about oxygen meter,

`h2_sensor` contains info about sensor used for hydrogen measurements in gas and liquid phase,

`irradiation_chamber` contains info about advanced irradiation chamber where irraiation was performed,

`irradiation_setup` contains info about irradiation setup where irradiation and measurements were performed,

`o2_h2_reactor` contains info about 4-neck photoreactor for O2 and H2 simultaneous measurements,

`power_meter` contains info about power meter and thermopile sensor used for adjusting LED power before photocatalytic tests,

`robust_probe_o2` contains info about sensor used for oxygen measurements in gas and liquid phase,

`unisense_h2_meter`contains info about amplifier for H2 sensor,

`vortexer` contains info about vortex mixer used for photocatalyst suspension preparation
